Mission · what Orbis does

mission

Orbis is a local-first AI design tool. Type what you imagine — or point it at any site you love — and watch it become a real, interactive design. Web and mobile, running live on your machine. No cloud required.

  • Imagine → Create

    From a prompt, watch it build

    Describe what you want and Orbis builds several complete, self-contained directions live — then previews any of them full-screen.

  • Imagine · Clone

    Clone any webpage — then make it yours

    Point Orbis at any URL — here, apple.com — and it rebuilds the page as a real, editable prototype, not a screenshot. Keep modifying it like anything you made from scratch.

  • Create · Edit

    Edit anything in the frame

    Select any element — copy, color, image, type, spacing — and change it inline, or ask AI to restyle a whole section. Every layer stays real and editable, and updates in place.

  • Create · Design system

    A design.md you can reuse

    Orbis distills the whole look — palette, type, spacing, motion, components — into a portable design.md. Drop it into any project and the agent builds to your system.

  • Ship → Figma

    Send it to Figma — fully editable

    One step converts your rendered web frames into a real Figma file: every layer, text style, component and asset intact and editable — not a flat screenshot. Plus journey maps and dev-ready handoff specs.

  • Ship · Handoff

    A dev-ready spec for any screen

    Every screen comes with a complete handoff: design tokens, type scale, colour, spacing redlines and component notes — measured from the real design, never estimated.
