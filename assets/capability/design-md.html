VYBEHAUS — Cyan Frequency

A dark, technical, glass-and-grid sound system. Ultra-thin Archivo display set against pure black, electric-cyan accents, and a breathing dot-matrix field — premium and atmospheric, built for a modern music brand.

Palette

  • Primary #67e8f9 — main accent: active states, play controls, links, equalizer, one hero emphasis per section.
  • Secondary #06b6d4 — supporting accent for "movements" emphasis and deeper cyan moments.
  • Tertiary #6c87fe — reserved violet-blue accent for contrast pops (collection glow, "memories").
  • Neutral #000000 — the full-bleed black foundation behind every surface.
  • Background #000000 — page canvas; never lifted to grey.
  • Surface #ffffff — primary button fills and the wordmark mark; used sparingly as pure white.
  • Muted rgba(255,255,255,0.58) — secondary body copy.
  • Subtle rgba(255,255,255,0.34) — captions, meta, disabled labels.
  • Border rgba(255,255,255,0.12) — hairline dividers and card edges.

Typography

  • Headline (display) — Archivo, weight 200, line-height 0.92–0.98, letter-spacing -0.01em. Fluid scale (min/preferred/max): display 36→80px, h1 48→120px, h2 32→64px, h3 24–28px. Italic weight 300 for accent words.
  • Body — Archivo, weight 400, 16–19px, line-height 1.5.
  • Label — Rajdhani, weight 600, 10–12px, letter-spacing 0.24–0.42em, uppercase. Used for eyebrows, buttons, meta, nav.
  • Numerics — tabular-nums applied via font-variant-numeric on all stats and timecodes.

Spacing & Layout

  • Base unit 8px. Scale: 1 · 8 · 12 · 16 · 20 · 24 · 32 · 40.
  • Section padding: 96–176px desktop, 96px tablet, 64px phone (py-24 → py-44).
  • Card padding: 32–56px.
  • Container max-width 1320px, full-bleed background. 12-column intent expressed via asymmetric 1.05fr / 0.95fr grids.
  • Gaps: 8 · 12 · 16 · 24px; card grids use 1px hairline gutters over a bg-white/5 grid.

Radii & Elevation

  • Border-radius: 0px everywhere (sharp corners are load-bearing). Only inner pills/dots may round.
  • Elevation: glass-first. Outer gradient hairline shell (1px padding, linear-gradient(135deg, rgba(103,232,249,.5), rgba(255,255,255,.08) 40%, rgba(255,255,255,.04))) wrapping a backdrop-blur(6px) glass surface.
  • Shadows: deep 0 0 80px rgba(0,0,0,.8); cyan glow 0 0 14px rgba(103,232,249,.45) on active controls only.

Motion

  • Personality: expressive but interface-focused — snappy-precise, never bouncy. Cyan glow blooms on activation.
  • Durations: 180ms (press), 300ms (hover/state), 700ms (reveal). Easing: cubic-bezier(0.4,0,0.2,1).
  • Signature: a full-bleed WebGL dot-matrix field with a slow breathing pulse and subtle pointer drift (DOM/CSS dot fallback).
  • Scroll: smooth scroll; sections reveal with a staggered 60–90ms opacity + 26px translate as they enter.
  • What animates: equalizer bars, waveforms, hover/press states, reveals, the particle field. What stays still: type, layout, spacing at rest. All motion ships a prefers-reduced-motion branch.

Components

  • Button — primary: white fill, black text, 0px radius, 15×26px padding, Rajdhani uppercase 0.28em; hover → cyan fill + glow.
  • Button — ghost: transparent, 1px rgba(255,255,255,.24) border; hover → cyan text + cyan border.
  • Card: gradient shell + glass surface, hairline grid edges, card-rise hover (translateY -4px + cyan edge).
  • Player / collection controls: square 56–64px, white→cyan or bordered→cyan; play/pause swap with active glow.
  • Input: transparent, 1px border, focus → cyan border; inline submit button in white→cyan.
  • Nav: transparent → black/70 frosted on scroll, hairline bottom border, mobile sheet.

Voice & Imagery

Confident, atmospheric, a touch editorial — short declaratives ("Where sound becomes identity."). Imagery is non-photographic: the generative dot-matrix field, animated waveforms/equalizers, and glass surfaces stand in for art. Iconography is linear (Solar-style), 1.5 stroke, rounded joins.

Usage

Paste this into the project root and tell the agent to use it as the active design system.

:root {
  --color-primary: #67e8f9;
  --color-secondary: #06b6d4;
  --color-tertiary: #6c87fe;
  --color-bg: #000000;
  --color-surface: #ffffff;
  --color-fg: #ffffff;
  --color-muted: rgba(255, 255, 255, 0.58);
  --color-border: rgba(255, 255, 255, 0.12);
  --font-display: 'Archivo', system-ui, sans-serif; /* weight 200 */
  --font-label: 'Rajdhani', system-ui, sans-serif; /* 600, uppercase, 0.3em */
  --radius: 0px;
  --ease: cubic-bezier(0.4, 0, 0.2, 1);
}
